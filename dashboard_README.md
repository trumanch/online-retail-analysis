# E-commerce Dashboard (Streamlit)

## Project Overview
This project is an interactive dashboard built with Streamlit to analyse online retail sales data.

## Features
- Country filter
- Revenue, orders, customers, and average order value KPIs
- Monthly revenue trend
- Top products by revenue
- Country-level revenue comparison

## Tools Used
- Python
- Pandas
- Streamlit

## Skills Demonstrated
- Dashboard development
- KPI reporting
- Interactive data exploration
- Business-focused data visualisation
