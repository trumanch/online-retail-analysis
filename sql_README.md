# SQL Business Analysis

## Project Overview
This project demonstrates SQL-based analysis of a retail dataset to answer key business questions and generate insights.

## Tools Used
- SQL (SQLite)
- Python
- Pandas

## Key Analyses
- Monthly revenue trends
- Top products by revenue
- Top customers by spending
- Revenue by country
- Average order value

## Key Findings
- Revenue trends vary across months, indicating seasonality
- A small number of products generate most revenue
- High-value customers contribute significantly to sales

## Skills Demonstrated
- SQL querying
- Aggregation and grouping
- Business analysis using data
- Data extraction and reporting
